仅用于个人存放代码，如果使用过程中出现任何问题，与本人无关

上传本地代码：
1.   git init                                          // 将此目录变为Git可以管理的仓库
2.   git add .                                      // 将所有文件添加到仓库
3.   git commit -m "提交文件"	  // 将所有文件提交到仓库，双引号内是提交注释。
4.   git remote add origin git@github.com:wangjiax9/practice.git  //关联远程仓库
5.   git push -u origin master	  //进行代码提交

更新本地代码：
1.   git add .                                      // 将所有文件添加到仓库
2.   git commit -m "提交文件"	  // 将所有文件提交到仓库，双引号内是提交注释。
3.   git push -u origin master	  //进行代码提交
